Demo - Testna Stranica

Kako otvoriti: Pokrenite dev server (npm run dev) i otvorite:
http://localhost:5173/widget-test.html

Dobrodošli na demo stranicu

Ova stranica služi za testiranje widgeta. Widget bi se trebao pojaviti u donjem desnom kutu.

O demo sustavu

Ovo je demo sustav za testiranje AI asistenta za gradove. Sustav je poznat po svojoj funkcionalnosti i jednostavnosti korištenja.

Kontakt

Za više informacija, molimo kontaktirajte gradsku upravu putem widgeta koji se nalazi u donjem desnom kutu stranice.

Usluge

Demo sustav nudi različite usluge za građane, uključujući informacije o gradskim službama, dokumentaciji i kontakt podacima.
